# 🆔 ID Validator Python 🐍

A simple Python project that allows users to enter and validate person IDs with:
- ✅ Length verification
- 🔢 Numeric format checking
- 🚫 Duplicate ID prevention
- 📦 Array-based storage using NumPy

---

## 📌 Features ✨

🔹 Ask the user for the number of people  
🔹 Enter personal IDs  
🔹 Check that each ID contains exactly **8 digits**  
🔹 Prevent repeated IDs  
🔹 Store IDs in a NumPy array  

---

## 🛠️ Technologies Used 💻

- 🐍 Python
- 🔢 NumPy
- 📚 Basic algorithms:
  - Searching
  - Loops
  - Input validation
